### `inferLabels`

```ts
export const inferLabels = (
  min: number,
  max: number,
  target = 5,
): number[] => …
```

Pick "nice" tick positions inside [min, max] aiming for roughly `target` ticks.
Uses 1× / 2× / 5× multiples of the nearest power of ten so labels read cleanly.
