Initial public release: Svelte 5 components for interactive math visualization. Includes `<Mafs>` root, `<Coordinates.Cartesian>`, the `<Plot>` namespace (`OfX`, `OfY`, `Parametric`, `Inequality`, `VectorField`), display primitives (`Point`, `Line`, `Circle`, `Ellipse`, `Polygon`, `Vector`, `Text`, `Transform`), `<MovablePoint>` with keyboard a11y, and pan/zoom + drag actions. Faithful API port of [Mafs](https://mafs.dev) (MIT) by Steven Petryk.
